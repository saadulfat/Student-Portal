Student Portal

Student Portal is a comprehensive dashboard for students to organize, learn, and track their academic activities in one place. The portal combines traditional productivity tools with AI-powered features to enhance the learning experience.

Key Features

  • Lessons: View and generate lessons, including AI-generated content for any topic.
  • Quizzes: Attempt scheduled class quizzes and generate practice quizzes using AI. Practice quizzes provide instant feedback and scoring.
  • Assignments: Submit scheduled assignments and practice with AI-generated assignments for self-assessment.
  • Records: Track your lesson history, quiz scores, and assignment submissions. Only scheduled/class activities are recorded for progress tracking.

    • How It Works

    • Backend: Built with Flask and MySQL, handling user authentication, data storage, and record management.
    • AI Integration: Uses OpenAI (via OpenRouter) to generate lessons, quizzes, and assignments on demand. AI-generated practice activities are for self-study and not stored in records.
    • Frontend: Responsive interface using HTML, CSS, PaperCSS, and JavaScript for a smooth user experience.

    Project Purpose

    The Student Portal aims to streamline academic management for students, combining essential tools and AI-powered learning in a single platform. It is designed to be extensible, with plans to add more APIs (e.g., Wikipedia, Dictionary, Language Translator) and fun features like games in the future.

    Future Plans

    • Integrate more APIs for reference and learning.
    • Add multiplayer and interactive games for engagement.
    • Enhance analytics and progress tracking.
    • Enable notifications and reminders for assignments and quizzes.
